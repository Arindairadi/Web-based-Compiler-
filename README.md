An online integrated compiler visualization platform was developed to address the "black box" nature of compilation processes in programming education, where internal stages such as lexical analysis, syntax parsing, and code generation remain hidden from learners. The platform supports nine programming languages (Python, PHP, JavaScript, R, Java, C, Swift, BrainFuck, and Go) with interactive 3D visualization of six compiler stages. The specific objectives were to: (1) analyze system requirements for a web-based compilation platform, (2) design the system for multi-language compilation and visualization, (3) develop the system, and (4) test and evaluate it for correctness and usability. The system was built using a three-tier architecture with HTML, CSS, JavaScript, and Three.js for the frontend, and PHP with MySQL for the backend, following an Agile methodology. A sequential pipeline algorithm processes source code through lexical, syntax, semantic, SSA, optimization, and assembly generation stages, each visualized in color-coded 3D. Key achievements include dual authentication (guest and registered users), compilation history, activity logging, and responsive design. The project was conducted over eight months (November 2025 – June 2026) at Kabale University, Uganda. Evaluation with approximately 100 users demonstrated that interactive visualization improves program understanding and debugging, while multi-language support facilitates knowledge transfer across languages. Limitations include simulated compiler stages and local evaluation. Future work will integrate real compilers, AI-based error explanation, and memory visualization.
